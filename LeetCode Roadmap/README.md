# Coding Personal Checklist

How to use each topic:
- Learn the concept first from one resource.
- Write small Python examples yourself.
- Then solve LeetCode problems from that topic.
- Save solutions in this repo.

---

## Topic Checklist

### 1. Arrays
- Learn indexing, iteration, slicing.
- Practice basic loops and element access.
- Solve easy array problems.

### 2. Strings
- Learn string indexing and slicing.
- Practice character counting and comparison.
- Solve easy string problems.

### 3. Hash Table
- Learn dict and set usage.
- Practice frequency counting.
- Solve lookup-based problems.

### 4. Two Pointers
- Learn left and right pointer patterns.
- Practice moving pointers safely.
- Solve basic pointer problems.

### 5. Sliding Window
- Learn fixed and variable window logic.
- Practice window expansion and shrink.
- Solve substring and subarray problems.

### 6. Stack
- Learn LIFO behavior.
- Practice using lists as stacks.
- Solve parentheses and undo-style problems.

### 7. Queue
- Learn FIFO behavior.
- Practice using deque.
- Solve breadth-style problems.

### 8. Binary Search
- Learn sorted array assumptions.
- Practice mid calculation and boundaries.
- Solve search and condition problems.

### 9. Trees
- Learn node, left, right concepts.
- Practice recursion and traversal.
- Solve traversal and depth problems.

### 10. Heap / Priority Queue
- Learn min heap and max heap.
- Practice top-K logic.
- Solve priority-based problems.

### 11. Graphs
- Learn nodes and edges.
- Practice DFS and BFS.
- Solve traversal problems.

### 12. Dynamic Programming
- Learn overlapping subproblems.
- Practice memoization and tabulation.
- Solve classic DP problems.

### 13. Greedy
- Learn local optimal choices.
- Practice decision-based logic.
- Solve interval and selection problems.

### 14. Sorting
- Learn built-in sorting and custom keys.
- Practice sorting patterns.
- Solve ordering problems.
